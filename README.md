# RESTE API Star Wars

## API desenvolvida para servir ao jogo da franquia Star Wars

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com), [Node.js](https://nodejs.org/en/). 

### 🎲 Como executar o Back End

```bash
# Clone este repositório
git clone <https://github.com/cipolli/starwars>

# Acesse esse dertório pelo termina \starwars\target
starwars-0.0.1-SNAPSHOT.jar

# A API inciará na porta:8080
acesse <http://localhost:8080>

#Também é possível usar um client como o insominia para testar os endpoints
